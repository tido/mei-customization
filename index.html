Welcome to Tido MEI Customization

A minimal MEI customization that defines a basic subset of elements with appropriate restrictions for basic use cases.

From this page you can access the generated documentation and the RNG schema file

The project also contains a test environment in Node.js and available as an npm package.

For more information regarding the internals please see the GitHub repository
